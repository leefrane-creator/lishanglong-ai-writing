# 李尚龙AI写作工作流

基于李尚龙分享的核心方法论，提炼为可执行的 AI 写作 skill。

## 核心理念

制造共鸣，而不是制造焦虑。AI 负责效率与角度，个人态度负责温度与辨识度，两者缺一不可。

## 五大心智模型

1. **情绪波动驱动传播** — 人们转发不是因为内容好，而是因为情绪被击中
2. **反常识即注意力** — 常识性内容没人看，反常识内容才值得传播
3. **个人态度过滤器** — AI 可以生成 100 个角度，但只有你能决定哪个值得写
4. **细节即辨识度** — 通用道理人人会写，具体细节才是你的护城河
5. **去AI味检查清单** — AI味 = 抽象词汇 + 排比句式 + 说教结尾

## 工作流

```
Step 0: 问题分类（有选题 / 无选题 / 有初稿需优化）
    ↓
Step 1: 选题筛选（情绪波动测试 · 三问法）
    ↓
Step 2: 反常识角度挖掘（AI生成5个角度 → 个人态度过滤）
    ↓
Step 3: AI生成大纲（情绪弧线 + 素材支撑检查）
    ↓
Step 4: AI出钩子稿（前3-5句话必须让人停下来）
    ↓
Step 5: 全文生成 + 去AI味处理
```

每一步结束后停下来，等用户确认再继续。
