# Prompt模板库 · 李尚龙AI写作工作流

## Step 1 选题筛选（情绪波动测试）

### 三问法Prompt
```
我要写的选题是：【XXX】

请帮我用以下三问评估这个选题：
1. 这个选题会让人又高兴又难受吗？（情绪波动是传播的底层逻辑）
2. 读者能在里面找到自己吗？（共鸣 > 焦虑）
3. 分享出去，读者会说"这说的就是我"，还是"这说的是别人"？

请逐条分析，最后给出结论：这个选题值得写吗？
```

## Step 2 反常识角度挖掘

### 基础版
```
关于【选题】，大多数人的主流观点是什么？
请给我5个反常识的、有争议性的角度，要求：
1）打破大众认知惯性；
2）有一定道理但绝不是鸡汤；
3）适合作为文章核心观点。
```

### 加强版（适合竞争激烈的选题）
```
关于【选题】：
1. 先告诉我网上最常见的3种写法和角度
2. 再给我5个"大家都没想到但仔细想很有道理"的反常识角度
3. 每个角度给一句核心金句，20字以内
```

### 个人态度过滤提示
```
以上5个角度中，请选出你最认同的一个。

判断标准：
- 这个角度你有没有真实经历支撑？
- 你能不能讲出一个具体的故事来证明这个观点？
- 如果让你和朋友聊这个话题，你会不会选择这个角度？

选好后告诉我：你选第几个？为什么？
```

## Step 3 大纲生成

### 公众号/头条长文版（1500-2000字）
```
请用这个核心观点【XXX】帮我搭一个公众号长文大纲。

结构要求：
- 开篇：具体场景或反常识判断作为钩子（100字以内）
- 第一段：建立认知冲突，让读者觉得"说得对但不对劲"
- 论点1/2/3：每个论点包含一个真实场景+一个核心判断
- 结尾：不说教，说感受，留一个开放性问题给读者

情绪弧线要求：由压抑/困惑 → 释然/觉醒

请输出完整大纲，每个部分标注预计字数。
```

### 小红书短图文版（500字以内）
```
请用这个核心观点【XXX】帮我搭一个小红书笔记大纲。

结构：
- 标题：数字+反常识判断（15字以内）
- 开头：1句话戳痛点
- 正文：3个小标题，每个小标题下2-3句话
- 结尾：1句行动建议+互动引导

请输出完整大纲。
```

### 大纲自审清单
```
大纲已生成，请帮我检查以下问题：

1. 逻辑自洽检查：
   - 论点之间是否有逻辑断层？
   - 从开头到结尾的推导是否顺畅？

2. 素材支撑检查：
   - 每个论点我有没有真实素材可以填？
   - 哪些论点需要补充具体案例？

3. 情绪弧线检查：
   - 整体情绪走向是向上的吗？
   - 结尾是否给读者留下希望或行动动力？

请输出检查结果，标注需要修改的地方。
```

## Step 4 钩子稿生成

### 标准钩子Prompt
```
根据这个大纲，帮我写开头钩子（前3-5句话）。

要求：
- 前3句必须让人停下来，不能滑走
- 用具体场景、数字或反常识判断开头
- 禁止鸡汤句式
- 禁止"你是否曾经..."这类模板开头
- 禁止"在这个XXX的时代..."这类背景铺垫
```

### 钩子类型参考

| 类型 | 示例 |
|------|------|
| 反常识判断 | "努力了10年，我终于发现努力是有上限的。" |
| 具体场景 | "上周三晚上11点，我盯着简历发呆了两小时。" |
| 数字冲击 | "全国有2000万人每天重复同一件没有意义的事，我是其中一个。" |
| 直接结论 | "我不建议你现在辞职，哪怕你真的很痛苦。" |
| 对话引入 | "'你为什么不早点告诉我？'朋友问我。我沉默了。" |

### 钩子优化Prompt
```
这个开头还不够抓人，请帮我优化。

优化方向（选一个）：
A. 更反常识 - 挑战读者的固有认知
B. 更具体 - 用细节让读者身临其境
C. 更冲突 - 制造悬念或矛盾
D. 更直接 - 去掉铺垫，直击痛点

请输出3个优化版本。
```

## Step 5 去AI味处理

### 去AI味检查清单

逐条检查，有问题立刻修改：

**词汇层**
- [ ] 删掉：总之、综上所述、不得不说、值得注意的是、毋庸置疑、显而易见
- [ ] 删掉：在某种程度上、从某种角度来看、不可否认的是
- [ ] 删掉：赋能、颠覆、生态、闭环、底层逻辑（除非真的需要）
- [ ] 删掉：非常、特别、十分、极其（程度副词过多）

**句式层**
- [ ] 不用排比三连："XXX，也是XXX，更是XXX"
- [ ] 不用说教式结尾："希望每个人都能..."
- [ ] 不用无实义感叹："这是多么令人感动啊！"
- [ ] 不用被动语态："被发现" → "我发现"

**内容层**
- [ ] 加入至少1个只有你才知道的细节
- [ ] 确认至少1处用了第一人称真实经历
- [ ] 结尾是感受，不是建议
- [ ] 每个论点都有具体案例支撑

**读感层**
- [ ] 通读一遍，找出"这句话不像你会说的"，改掉
- [ ] 找一个不了解你的朋友读开头3句，问他想不想继续看
- [ ] 大声读一遍，不顺口的地方改掉

### 去AI味改写Prompt
```
请帮我改写以下段落，去掉AI味：

【原文】

改写要求：
1. 删掉所有抽象词汇和套话
2. 换成具体场景或个人经历
3. 用口语化的短句
4. 保留核心观点，但用自己的语气表达

请输出改写后的版本。
```

## 选题灵感触发器

当没有选题时，用以下问题逼出素材：

1. 最近有没有一件事让你既开心又难过？
2. 身边有没有人做了一件让你佩服但你不会做的事？
3. 有没有一个你以前深信不疑、后来发现是错的观点？
4. 有没有一件事你做了很久才发现做法完全错了？
5. 最近有没有让你睡不着觉的问题？
6. 你有没有一个"别人都觉得正常，但你觉得不对劲"的观察？
7. 最近有没有人和你说过一句让你印象深刻的话？
8. 你有没有一个"如果早点知道就好了"的领悟？

## 平台适配指南

### 微信公众号
- 字数：1500-2500字
- 结构：故事开头 → 3-4个论点 → 情感升华结尾
- 特点：可以深度，注重逻辑和情感并重

### 今日头条/头条号
- 字数：800-1500字
- 结构：争议性开头 → 快速展开 → 明确结论
- 特点：标题要抓人，节奏要快，观点要鲜明

### 小红书
- 字数：300-800字
- 结构：数字标题 → 痛点开头 → 3个要点 → 行动建议
- 特点：视觉化、可扫描、实用性强

### 知乎
- 字数：2000字以上
- 结构：问题拆解 → 多角度分析 → 深度结论
- 特点：逻辑严密、论据充分、专业度高

### B站专栏
- 字数：1500-3000字
- 结构：个人经历 → 观点输出 → 互动引导
- 特点：语气轻松、像朋友聊天、多图配合
